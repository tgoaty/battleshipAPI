TASK:
Разработать REST API сервис для игры в “Морской бой”. Должна быть возможность создавать игру,
 обрабатывать ходы игроков и выявлять результаты игры. Нужно использовать websocket-ы для реализации комнаты

Нужно реализовать следующие эндпоинты:

1) POST /players/register
Регистрация игрока в системе

1.2) POST /players/login
Авторизация игрока

2) GET /players
Получение всех доступных (которые на данный момент не играют) игроков

3) POST /games/create
Создание игры. Метод принимает игроков и привязывает их в новой игре. Создает доску с рандомным
 расположением (но по правилам) кораблей и возвращает ее

4) GET /games
Получение всех активных игр с досками

5) WS /games/{game_sid}/play
Websocket – ручка для игры (комната)

Внутри комнаты нужно обработать сообщения для начала и завершения игры, ходы игроков
и выявление победившего игрока. Результат  игры нужно сохранить

6) GET /players/{player_sid}/stats
Получение списка всех игр с датой и результатами
